### Run ubuntu setup script

#### 24.04

```bash
sudo apt-get update && sudo apt-get install -y curl && curl -fsSL https://raw.githubusercontent.com/kiki-kanri/linux-configs-and-scripts/refs/heads/main/ubuntu/24.04/setup.sh | sudo bash
```

> [!IMPORTANT]
> For use with brand-new systems only!
